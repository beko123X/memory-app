# 📸 Memories App - Full Stack MERN Application

A social media application that allows users to post, update, and delete their special memories. This project features full authentication (JWT & Google OAuth), photo uploads, and a responsive design.

## 🚀 Live Demo
[Check out the Live Site on Netlify](https://memories-gallary.netlify.app/)

---

## ✨ Features
- **User Authentication:** Secure login and sign-up using JWT (JSON Web Tokens) or Google OAuth.
- **Post Management:** Users can create, edit, delete, and like posts.
- **Photo Uploads:** Integrated image handling using Base64.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.
- **Search & Filter:** Search for memories by title or tags (using specialized search logic).
- **Comments:** Real-time-like interaction for users to comment on posts.

---

## 🛠️ Tech Stack
- **Frontend:** React.js, Redux, Material-UI, Axios.
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB Atlas.
- **Hosting:** Netlify (Frontend) & Pxxl.app (Backend).
